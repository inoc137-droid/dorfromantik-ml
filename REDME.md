\# Dorfromantik ML



Dieses Projekt beschäftigt sich mit der algorithmischen Modellierung und Analyse des Spiels \*Dorfromantik\* mit dem Ziel, Machine Learning Ansätze zur Entscheidungsfindung zu entwickeln.



\## 🚀 Ziele



\- Simulation von Dorfromantik-Spielzuständen

\- Bewertung von Spielzügen (Heuristiken / ML)

\- Optimierung von Platzierungsentscheidungen

\- Feature-Engineering auf Basis von Spielmechaniken (z. B. zusammenhängende Gebiete)



\## 🧠 Kernideen



\- Hex-Grid basiertes Board

\- Tile-basierte Spielmechanik mit Rotationen

\- DSU (Disjoint Set Union) zur Verwaltung von zusammenhängenden Gebieten

\- Frontier-basierte Platzierungslogik

\- Erweiterbarkeit für Sonderregeln (z. B. Wolken-Tiles)
